# EHR Analysis

## Scope

Workflow for electronic health record (EHR) predictive modeling using PyHealth, covering dataset loading, task definition, model training, evaluation, and clinical interpretation.

## Typical Inputs

- EHR datasets (MIMIC-III, MIMIC-IV, eICU, OMOP-CDM, or custom)
- patient visit tables (diagnoses, procedures, prescriptions, lab results)
- medical code mappings and ontologies

## Typical Outputs

- trained clinical predictive models
- evaluation metrics and calibration results
- ranked predictions and interpretability summaries

## Conceptual Seeds

- EHR predictive modeling pipelines
- clinical task definition and patient cohort construction
- medical code normalization and cross-system mapping patterns
- deep learning model training and evaluation for healthcare

- `pyhealth.medcode`
- `pyhealth.calib`
