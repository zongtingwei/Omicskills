# SPR/BLI Binding Characterization

## Scope

SPR and BLI assay planning, kinetic interpretation, and troubleshooting guidance. Use when: (1) Planning binding kinetics experiments, (2) Troubleshooting poor/no binding signal, (3) Interpreting kinetic data artifacts, (4) Choosing between SPR vs BLI platforms.

## Typical Inputs

- target protein structure, sequence, accession, or campaign brief
- design constraints, validation questions, or ranking criteria
- upstream protein-design outputs when this skill is used mid-pipeline

## Typical Outputs

- tool-specific designs, predictions, search results, or planning artifacts
- decision-ready summaries for the next protein-design step
- handoff notes to downstream design, prediction, assay, or QC skills

## Conceptual Seeds

- protein-design workflow orchestration
- spr-bli-binding-characterization task patterns
- experiment-ready handoff design
