# ChIP Seq

## Scope

Workflow for ChIP-seq QC, peak calling, annotation, motif analysis, differential binding, and track visualization.

## Typical Inputs

- aligned treatment and control reads
- reference genome
- annotations

## Typical Outputs

- peak files
- annotated regions
- motif and differential binding summaries

## Conceptual Seeds

- ChIP-seq workflows
- peak-centric chromatin binding patterns
- signal track and coverage visualization patterns

## Optional Supplements

- `deeptools`
- `pysam`
