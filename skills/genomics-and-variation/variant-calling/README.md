# Variant Calling

## Scope

Workflow for small-variant and structural-variant discovery, filtering, annotation, and interpretation from sequencing data.

## Typical Inputs

- aligned reads
- reference genome
- optional truth set or panel resources

## Typical Outputs

- VCF files
- filtered variant tables
- annotation summaries

## Conceptual Seeds

- variant-calling workflows
- clinical and research variant interpretation patterns
- alignment and VCF handling patterns
