# Bulk RNA Expression

## Scope

Python-first workflow for bulk RNA-seq expression intake, normalization, sample QC, and downstream-ready matrices.

## Typical Inputs

- count matrix
- sample metadata
- optional BAM or quantification outputs

## Typical Outputs

- normalized matrix
- sample QC plots
- PCA or clustering summaries

## Conceptual Seeds

- bulk transcriptomics matrix handling patterns
- RNA quantification workflows
- bulk RNA QC and expression summarization patterns

## Optional Supplements

- `pydeseq2`
- `pysam`
