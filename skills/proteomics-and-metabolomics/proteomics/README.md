# Proteomics

## Scope

Workflow for mass spectrometry proteomics including identification, inference, quantification, QC, differential abundance, and PTM follow-up.

## Typical Inputs

- raw MS files or converted spectra
- sample metadata
- sequence database

## Typical Outputs

- peptide or protein tables
- QC summaries
- differential abundance results

## Conceptual Seeds

- proteomics workflows
- protein quantification and QC patterns
- Python mass spectrometry tooling patterns
