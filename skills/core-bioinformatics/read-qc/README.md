# Read QC

## Scope

Workflow for sequencing read QC, trimming, contamination screening, and pre-alignment cleanup.

## Typical Inputs

- raw FASTQ files
- adapter sequences
- optional sequencing metadata

## Typical Outputs

- QC reports
- filtered or trimmed reads
- contamination summaries
