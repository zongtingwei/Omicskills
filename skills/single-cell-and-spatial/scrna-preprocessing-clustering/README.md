# scRNA Preprocessing And Clustering

## Scope

Python-first single-cell RNA workflow for QC, normalization, dimensionality reduction, neighborhood graph construction, and clustering.

## Typical Inputs

- AnnData object or count matrix
- cell metadata
- gene annotations

## Typical Outputs

- filtered single-cell object
- UMAP or PCA embeddings
- cluster assignments

## Conceptual Seeds

- single-cell preprocessing workflows
- single-cell clustering workflows
- scanpy-style analysis patterns

## Optional Supplements

- `anndata`
- `scanpy`
