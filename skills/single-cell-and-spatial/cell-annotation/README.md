# Cell Annotation

## Scope

Workflow for single-cell cell type annotation using marker review, reference mapping, and classifier-based labeling with confidence checks.

## Typical Inputs

- clustered single-cell object
- marker gene sets
- optional reference atlas

## Typical Outputs

- cell type labels
- cluster marker summaries
- confidence or ambiguity flags

## Conceptual Seeds

- single-cell annotation workflows
- reference-based cell labeling patterns
- scanpy-centered single-cell interpretation patterns

## Optional Supplements

- `scanpy`
- `scvi-tools`
