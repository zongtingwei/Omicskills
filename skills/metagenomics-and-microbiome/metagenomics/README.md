# Metagenomics

## Scope

Workflow for shotgun metagenomics including abundance profiling, functional profiling, AMR detection, and strain tracking.

## Typical Inputs

- metagenomic FASTQ files
- sample metadata
- reference databases

## Typical Outputs

- taxonomic abundance tables
- functional profiles
- AMR or strain summaries

## Conceptual Seeds

- shotgun metagenomics workflows
- community profiling and AMR task patterns
