# Source Mapping

This repository was curated from three broad source types:

- a structured omics workflow taxonomy
- a large biomedical leaf-skill collection
- a scientific tooling and database skill collection

Those upstream sources are not required at runtime. This repository is intended to be self-contained.

## Group Mapping

### `transcriptomics`

- Primary role: bulk expression and RNA workflow taxonomy
- Secondary role: RNA-specific leaf workflows and QC variants
- Supplement role: count-modeling, sequence handling, and expression-oriented tool references

### `single-cell-and-spatial`

- Primary role: single-cell and spatial workflow taxonomy
- Secondary role: fine-grained cell annotation and spatial task variants
- Supplement role: single-cell Python ecosystem references

### `epigenomics-and-regulation`

- Primary role: chromatin, methylation, and regulatory workflow taxonomy
- Secondary role: ATAC- and ChIP-focused leaf workflows
- Supplement role: coverage visualization, file handling, and network-inference references

### `genomics-and-variation`

- Primary role: variant, CNV, long-read, assembly, and comparative workflow taxonomy
- Secondary role: clinically flavored and long-read-focused leaf workflows
- Supplement role: alignment, VCF, and annotation tool references

### `metagenomics-and-microbiome`

- Primary role: community profiling and microbial comparative workflow taxonomy
- Secondary role: metagenomic, microbiome, and surveillance leaf workflows
- Supplement role: ecological and phylogenetic tooling references

### `proteomics-and-metabolomics`

- Primary role: proteomics, metabolomics, and structural analysis taxonomy
- Secondary role: mass-spectrometry and imaging leaf workflows
- Supplement role: Python MS tooling and structure/database references

### `multi-omics-and-systems`

- Primary role: integration, pathways, systems models, and causal inference taxonomy
- Secondary role: broad cross-modal analysis patterns
- Supplement role: pathway, interaction, and modeling references

### `core-bioinformatics`

- Primary role: foundational sequence, alignment, QC, reporting, and workflow patterns
- Secondary role: file handling and alignment utility patterns
- Supplement role: literature, plotting, and database-access references
